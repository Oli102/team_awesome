# team_awesome

John - Detection Lead — owns detection.js. Responsible for landmark math, distance calculations, and raw signal extraction. This person translates FaceMesh data into the numbers your state engine needs.

AJ - State Engine Lead — owns stateEngine.js and config.js. Responsible for mapping raw signals to named states using thresholds, timing, and hysteresis. This person is the bridge between sensing and meaning.
